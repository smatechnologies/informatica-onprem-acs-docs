# Release notes

There are no files to create, job subtypes to add, or Java software to install. All information required for the integration is contained in the Informatica On-Prem ACS connector. The connector contains the dynamic screen definitions using JSON schemas that are used to display the connection and task definition requirements.

## Winter 25

#### 25.0.0

2025 January

**Added the Informatica On-Prem ACS connector.** The connector provides access to Informatica On-prem environment workflows via the ACS (Agentless Connector System) framework.
